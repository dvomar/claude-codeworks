# Frontend Design

Frontend design skill for creating distinctive, production-grade UI components with high design quality that avoids generic AI aesthetics.

## What's included

### Skills
- `/frontend-design` — Guides creation of polished web components, pages, and applications with creative, non-generic styling

## Installation

```
/plugin marketplace add dvomar/claude-codeworks
/plugin install frontend-design@codeworks
```

## Usage

Build a UI component with high design quality:
```
/frontend-design Build a pricing page with three tiers
```

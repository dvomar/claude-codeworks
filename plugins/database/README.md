# Database

Database schema, query, and migration reviewer. Checks performance, security, and correctness of SQL and ORM usage.

## What's included

### Agents
- **database-reviewer** — Reviews schemas, queries, migrations, and ORM usage for performance, security, and correctness

## Installation

```
/plugin marketplace add dvomar/claude-codeworks
/plugin install database@codeworks
```

## Usage

The database-reviewer agent activates automatically when working with database-related code. You can also invoke it directly:

```
Review the database schema and queries in this project for performance and security issues.
```
